# Service Blueprints

Tools for generating service blueprint diagrams from blueprint contract data.

| Tool | Description |
|---|---|
| [Figma plugin](figma/README.md) | Generates native Figma frames and components that designers can edit |
| SVG renderer (`render-svg.js`) | Renders a local SVG preview — no Figma needed |

Run `node build.js` to build the Figma plugin and render the SVG in one step.
